Detailed Description

template<typename NoiseModel>
class qpp::QNoisyEngine< NoiseModel >

Noisy quantum circuit engine, executes qpp::QCircuit.

See also
qpp::QCircuit, qpp::NoiseBase

Assumes an uncorrelated noise model that is applied to each non-measured qubit before every step in the logical circuit

Template Parameters
NoiseModelQuantum noise model, should implement at least NoiseModel::get_d() and the functor qpp::cmat NoiseModel::operator()(qpp::cmat, qpp::idx). I recommend to use a noise model derived from qpp::NoiseBase.
